# How to Redirect Gatsby URLs

Use `jsÃ·actions.createRedirect()` in `gatsby-node.js` like so

```js
exports.createPages = ({ actions }) => {
  actions.createRedirect({
		fromPath: `/`,
		toPath: `/fr`,
		isPermanent: true,
	})
}
```
